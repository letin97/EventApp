# Ứng dụng ăn theo sự kiện lớn

Ứng dụng cập nhật các sự kiện lớn diễn ra trong nước, thông tin các các ngày lễ hội, các ngày kỉ niệm lớn trong năm.

## Yêu cầu thiết bị
-Android studio : 3.0.1

-Gradle : 4.1

-Min sdk : 19

-Target sdk : 26

-Language : Java
